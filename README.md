# Usage
You can move hexagons with keys: **Q**, **W**, **E**, **A**, **S**, **D**
| Key       | Direction       |
| ----------------- | ----------------- |
| **Q** | Top-left |
| **W**    | Top |
| **E**    | Top-right |
| **A**    | Bottom-left |
| **S**    | Bottom |
| **D**    | Bottom-right |

# Examples
