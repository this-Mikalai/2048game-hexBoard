# 2048game-hexBoard

Demo: http://this.mikalai-2048hex.surge.sh

# Usage
You can move hexagons with keys: **Q**, **W**, **E**, **A**, **S**, **D**
| Key       | Direction       |
| ----------------- | ----------------- |
| **Q** | Top-left |
| **W**    | Top |
| **E**    | Top-right |
| **A**    | Bottom-left |
| **S**    | Bottom |
| **D**    | Bottom-right |

# Examples
You can also change the radius of the hexagon. Value 2 is default.
